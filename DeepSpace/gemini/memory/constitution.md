# Resumo do Projeto — KingPong

_Atualizado em: 29/05/2026_

## Stack
- Linguagem: TypeScript
- Framework: React Native
- Banco de dados: SQLite
- Build/Lint: ESLint, Prettier, tsc

## Arquitetura
Feature-based Architecture com foco em componentes de UI e lógica de jogo encapsulados por funcionalidade.

## Convenções de Código
- Nomenclatura: PascalCase para arquivos e componentes.
- Estilo: Padrão da comunidade React Native.
- Comentários: Código autoexplicativo (zero comentários).
- Erros: Try/catch nativo.

## Testes
- Foco: E2E com Detox.
- Cobertura Meta: 90% dos fluxos críticos.
- Localização: Pasta `/e2e` na raiz.

## Processo (Modo POC)
- Sem Git ou CI no momento.
- Foco em validação de gameplay.
